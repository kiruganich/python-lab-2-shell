# Расширенная CLI оболочка (Medium уровень)

Этот проект представляет собой расширенную командную оболочку (shell) на Python с функционалом работы с архивами, поиском по содержимому файлов, историей команд и отменой действий.

## Функционал

### Основные команды (Easy уровень)
- `ls [-l] [path]` - Отображение содержимого директории
- `cd [path]` - Изменение текущей директории
- `cat <file>` - Вывод содержимого файла
- `cp [-r] <source> <destination>` - Копирование файлов и директорий
- `mv <source> <destination>` - Перемещение или переименование файлов и директорий
- `rm [-r] [-f] <path>` - Удаление файлов и директорий

### Дополнительные команды (Medium уровень)
- `zip <folder> <archive.zip>` - Создание ZIP-архива из папки
- `unzip <archive.zip> [extract_path]` - Распаковка ZIP-архива
- `tar <folder> <archive.tar.gz>` - Создание TAR.GZ архива из папки
- `untar <archive.tar.gz> [extract_path]` - Распаковка TAR.GZ архива
- `grep <pattern> <path> [-r] [-i]` - Поиск строк в файлах по шаблону
  - `-r` - Рекурсивный поиск в поддиректориях
  - `-i` - Игнорировать регистр при поиске
- `history [N]` - Просмотр истории команд (последние N команд)
- `clear_history` - Очистка истории команд
- `undo` - Отмена последней операции (cp, mv, rm, zip, tar)
- `exit` / `quit` - Выход из оболочки

### Особенности
- Автодополнение команд и путей с помощью Tab
- Логгирование всех действий в файл `logs/shell.log`
- Поддержка корзины для отмены удаления файлов
- Хранение истории команд между запусками программы
- Безопасная работа с файловой системой с проверками прав и существования путей
